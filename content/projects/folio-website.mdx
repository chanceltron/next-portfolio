# **Customizable Portfolio Website Template**

## **Overview**

This project is a versatile and customizable single-page application (SPA) designed as a basic template for a portfolio website. The website is built using HTML, CSS, and JavaScript, featuring a fully functional navigation bar, a light and dark theme switcher, a search bar for project filtering, and a carousel for review navigation. This template can easily be tailored and utilized by various professionals such as developers, photographers, and graphic designers to showcase their work.

## **Features**

- **Navigation Bar:** The website includes a functioning navigation bar that opens full screen modals, providing an intuitive and seamless user experience.

- **Theme Switcher:** A built-in light and dark theme switcher enables users to adjust the website's visual theme according to their preference, enhancing accessibility and user experience.

- **Search Functionality:** The search bar allows users to filter projects from the list based on keywords, making it easy to navigate through various projects and find relevant work.

- **Carousel:** The website features a carousel to navigate through the review section, providing an engaging and interactive way to showcase reviews or testimonials.

- **Customizability:** Designed as a basic template, the website can be easily customized to fit individual needs and preferences, making it an ideal starting point for various professionals to build their portfolio.

## **Technical Details**

This portfolio website template was developed using:

- **HTML:** Used HTML to create a well-structured foundation for the website.

- **CSS:** Utilized CSS for designing and styling the website, ensuring a visually appealing and user-friendly interface.

- **JavaScript:** Implemented JavaScript to add dynamic functionality such as the navigation bar, theme switcher, project filtering, and the review carousel.
